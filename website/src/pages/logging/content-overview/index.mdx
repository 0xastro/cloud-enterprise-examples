IBM Cloud Log Analysis with LogDNA 

</PageDescription>

Log Analysis with LogDNA provides a way for quickly finding the sources of issues while gaining deep insights into your cloud applications.
LogDNA provides the aggregation of event logging from your applications and services within the IBM Cloud environment. 

With LogDNA you have the ability to send logging events to an IBM Cloud Object Storage location for centralized management of events. LogDNA provides retention periods up to 7, 14, or 30 days of log searches. IBM Cloud Log Analysis with LogDNA supports long-term logging event storage and policies that can be defined to maintain log integrity and prevention of accidental deletion.
